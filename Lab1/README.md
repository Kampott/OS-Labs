# Lab1
## Вариант 12
Перемещает из папки 1 в папку 2 все файлы, возраст которых превышает 10 минут, а из папки 2 в папку 1 — перемещает файлы, возраст которых меньше 10 минут
Интеравл: 30 секунд.
### Запуск:
- Компилируем файл с помощью `sudo ./build.sh` - важно: необходимо иметь право на запуск (`chmod +x build.sh`).
- Записываем путь к папкам в файле config.txt, можно так-же задать интервал работы в секундах (по дефолту 30 секунд).
- Запускаем скрипт с помощью: `sudo ./mydaemon config.txt`

### Важные замечания:
Желательно изменять config.txt с помощью nano или vim - если появляются лишние невидимые символы в конфиге, демон не сможет прочитать конфиг.
